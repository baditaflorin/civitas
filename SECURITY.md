# Security Policy

## Supported Versions

Security fixes are accepted for the latest tagged release and the `main` branch.

## Reporting a Vulnerability

Please report vulnerabilities by opening a private security advisory at:

https://github.com/baditaflorin/civitas/security/advisories/new

If you cannot use GitHub advisories, contact the maintainer via the public profile at:

https://github.com/baditaflorin

Do not attach private evidence dumps, credentials, or sensitive personal data to public issues.
