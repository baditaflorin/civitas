# Code of Conduct

We want Civitas to be useful to civic investigators, journalists, researchers, and public-interest technologists across backgrounds and jurisdictions.

Be respectful, assume good intent, and keep discussions focused on evidence, implementation, and safety. Harassment, abuse, doxxing, threats, or discriminatory conduct are not acceptable.

Report conduct issues privately through:

https://github.com/baditaflorin/civitas/security/advisories/new
